WRITING

This is a user interface designed for an in-wall and in-table touchscreen with an optional analog interface. The UI was designed for SL Green headquarters in New York, and was designed to be installed by a third party using the Crestron content management system. The UI is tied to the content management system with displays in multiple areas; each screen and conference room has a separate controller for local control.

 

The SL Green account is held by Sensory Interactive.

This user interface was designed for St. Luke's Hospital in Kansas City. The client built an interactive video wall and wanted an interface that would allow multiple users to investigate the history of the hospital. This UI was developed using Photoshop, Illustrator, and After Effects. It was implemented on the CoolSigns content management system.

 

The early part of the video shows the Saint Luke's Foundation donor wall, which I also worked on for this project, doing initial design and proof-of-concept.

 

The St. Luke's Hospital account is held by Sensory Interactive.

The Parata account is held by Sensory Interactive (formerly by  Bluewhale Studios).
